推荐的文件存放逻辑：
```text
📂 资源文件夹 
 │
 ├─ 📂 漫画 (Comics)
 │   ├─ 📂/📦 [漫画1]        <-- 结构A: 多章节 (内含分卷文件夹)
 │   │   ├─ 📂  第1卷
 │   │   └─ 📂  第2卷
 │   │
 │   └─ 📂/📦 [漫画2]      <-- 结构B: 单章节 (内含图片)
 │       ├─ 📄 01.jpg
 │       └─ 📄 02.png
 │
 ├─ 📂 小说 (Novels)
 │   ├─ 📄 小说1.txt       (需手动设置封面)
 │   └─ 📄 小说2.epub      (自动读取封面)
 │
 └─ 📂 歌曲 (Music)
     ├─ 🎵 歌曲1.mp3     (单曲直接存放)
     └─ 📂 [周杰伦专辑]    (专辑文件夹)
         ├─ 🎵 歌曲文件.mp3
         ├─ 🖼️ cover.jpg
         └─ 📄 歌词.lrc
```
选定漫画/小说/歌曲为扫描的目标文件夹。

> **一个现代化、高颜值的 Android 漫画/小说/音频阅读器，支持安卓5-15，不过过低版本比如7以下可能有不可预测的报错**
> 基于 Jetpack Compose 构建，打造极致流畅的沉浸式阅读体验

## ✨ 核心特性 | Features

---

### 📖 漫画阅读器

- **多格式支持**：直读 ZIP / CBZ / RAR / CBR / PDF 以及文件夹内图片，无需解压
- **智能章节检测**：自动识别压缩包内的多级目录结构，按章节分卷展示
- **流畅翻页**：基于 LazyColumn 的高性能列表渲染，长篇漫画也不卡顿
- **阅读进度**：自动记录每部漫画的阅读位置（章节 + 页码），下次打开直接续读
- **沉浸模式**：全屏阅读，手势呼出控制条，支持快速跳转章节和页码

### 📝 小说阅读器

- **格式支持**：TXT 纯文本 + EPUB 电子书（解析目录、内嵌封面、HTML 章节内容）
- **排版定制**：
  - 字号 (12-36sp)、行高 (1.2-2.5x)、段间距 (8-32dp)、左右边距 (8-32dp) 全部可调
  - 4 种字体切换：系统字体 / 宋体 / 黑体 / 等宽
  - 字体颜色自定义：预设黑/深灰/灰/白色 + RGB 自定义调色
- **护眼主题**：5 种阅读背景色 — 羊皮纸 / 夜间模式 / 纯白 / 灰度 / 少女粉
- **自定义背景**：设置个性化背景图片 + 可调遮罩透明度
- **智能章节解析**：TXT 自动按正则匹配章节标题，EPUB 完整保留原书目录结构
- **阅读进度**：记录章节 + 滚动位置，续读无缝衔接

### 🎵 音频播放器

- **后台播放**：基于 Media3 / ExoPlayer 的后台播放服务，支持通知栏控制
- **LRC 同步歌词**：自动匹配 .lrc 歌词文件，滚动高亮当前歌词行
- **悬浮歌词**：全局悬浮窗歌词显示，多任务时也能看到歌词；双击回到播放界面，长按显示关闭按钮
- **黑胶唱片动画**：专辑封面以旋转黑胶唱片形式展示，播放时转动，暂停时停止
- **歌词 & 悬浮歌词颜色**：6 种预设颜色（白色/天蓝/荧光绿/淡黄/淡紫/黑色），各自独立配置
- **播放速度**：0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x 六档
- **自定义背景**：为播放器单独设置背景图 + 遮罩透明度
- **MeteorSlider 进度条**：同漫画阅读器，流星粒子效果
- **专辑模式 / 单曲模式**：音频显示可切换为专辑卡片模式或纯单曲列表平铺模式
- **单曲收藏**：支持对专辑内的单首歌曲独立收藏
- **MeteorSlider 流星进度条**：独创的彩色流星进度条，拖动时带有色彩流动的视觉效果

## 🏠 主页 & 管理功能

### 🎨 主题与个性化

| 主题 | 风格 |
|:---|:---|
| 🌸 **落樱** | 粉色系温柔少女风 |
| 🌙 **暗黑** | 赛博朋克暗色主题（跟随系统夜间模式自动切换） |
| 🖌️ **水墨** | 深色调沉稳风格 |
| 🍵 **抹茶** | 绿色系清新自然风 |

- 🌐 **中英双语**：中文 / English 一键切换，根据系统语言自动识别默认语言

### � 文件管理

- **多视图模式**：列表 / 3 列 / 4 列 / 5 列密集视图自由切换
- **多种排序**：最近阅读 / 最早阅读 / 名称正序 / 名称倒序
- **搜索**：实时搜索过滤库中项目
- **收藏**：一键收藏，快速找到喜欢的内容
- **多选批量操作**：长按进入多选模式，批量删除
- **重命名**：直接在 App 内修改名称
- **NSFW 标记**：将敏感内容标记为 NSFW
- **手动设置封面**：从设备相册选取图片作为自定义封面

### 📡 扫描与导入

- **目录扫描**：选定文件夹一键扫描，自动发现漫画/小说/音频文件
- **增量扫描**：已扫描且未修改的文件会被跳过，大幅节省重复扫描时间
- **外部打开**：从文件管理器直接用本App打开支持的文件（ACTION_VIEW Intent）
- **智能导入**：外部打开的文件自动复制到用户指定的持久存储目录，确保跨会话访问
- **封面自动提取**：
  - 漫画：从 ZIP/RAR/PDF 中提取第一页作为封面
  - EPUB：读取内嵌封面图片
  - 音频：提取 MP3/FLAC 等文件的内嵌专辑封面
  - 文件夹封面：自动使用目录中第一张图片
- **分档封面压缩**：根据库中项目数量智能调节封面分辨率和压缩质量，平衡清晰度与缓存大小

### 🔐 安全

- **生物识别解锁**：支持指纹识别保护 App 访问

## 🛠️ 强大的技术栈 (Tech Stack)

本项目完全采用现代 Android 开发技术构建：

| 库/工具 | 用途 |
| :--- | :--- |
| **Jetpack Compose** | 100% 声明式 UI，Material 3 设计规范 |
| **Coil** | 高性能异步图片加载 + 缓存 |
| **Media3 / ExoPlayer** | 后台音频播放核心 |
| **DocumentFile + SAF** | 安全的文件访问框架，跨应用文件读写 |
| **Biometric** | 生物识别（指纹/面部）安全认证 |
| **Junrar** | 原生 RAR 压缩包内容直读 |
| **JUniversalChardet** | 智能文本编码检测，告别乱码 |
| **PdfRenderer** | Android 原生 PDF 页面渲染 |
| **Gson** | JSON 序列化/反序列化（偏好存储） |
